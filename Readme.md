#  Hamburger Sidebar Menu

A simple and responsive **Hamburger Menu with Sidebar Toggle** built using **HTML, CSS, and JavaScript**.  
When the hamburger icon is clicked, a sidebar smoothly opens and closes.

##  Features
- Responsive hamburger menu
- Smooth sidebar open/close animation
- Built with pure HTML, CSS, and JavaScript
- Beginner-friendly and clean code
- Works on all screen sizes

##  Technologies Used
- HTML
- CSS
- JavaScript

##  Project Use Case
This project is useful for:
- Navigation menus
- Dashboards
- Mobile-first layouts
- Beginner JavaScript practice

##  Preview
Click the hamburger icon to open and close the sidebar.

##  How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. Click the hamburger icon to toggle the sidebar
